## 10. 文案风格备注

| 规则 | 说明 |
|------|------|
| 标题 / 卡片名 | 英文用 Title Case 或 uppercase；中文简洁名词化 |
| 数据单位 | 英文 `items` / `paid` / `pending`；中文「项」「已付」「待付」 |
| Offer | 中英文均保留 **Offer**（行业通用） |
| 月份 | EN: `Jun 2026` · ZH: `2026年6月` |
| 强调 | Welcome 摘要中数字加粗，中英文均保留 `<b>` 强调 |
