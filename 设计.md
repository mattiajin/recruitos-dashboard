# RecruitOS Dashboard — UI 设计规范

> **版本** v1.0 · **生产主题** Vivid Sky · **适用范围** New Design Tab  
> 本文是 Specs Tab 的完整离线扩展版，中文为主、术语保留英文。

---

## 1. 设计原则（Principles）

### 1.1 Structured clarity 结构化清晰

- 用 **Card 边界** 划分信息区，避免整页平铺大表
- Dashboard 采用 **Welcome → KPI → Bento** 三层递进
- 同排卡片共享行高，视觉节奏稳定

### 1.2 扁平近白（Flat near-white）

- 页面与卡片表面均为近白（`#FFFFFF`）
- **默认无卡片投影**；层次靠 1px 边框 + 背景 tint 区分
- Hover 以 **边框品牌色 tint** 为主，不做堆叠 elevation

### 1.3 单色系图表 + 一处强调色

- 图表主色：蓝色渐变 ramp（`chart-1` → `chart-4`）
- **关键数据点** 使用 coral-orange accent（`#EA6C2E`）
- Teal / Amber / Coral **仅用于 Status Tag**，不得作为图表扇区填色

### 1.4 适度密度（Moderate density）

- 总览页用 **Comfortable** 模式（见 §6）
- 列表/模块页预留 **Compact** 规则，待后续页面落地

### 1.5 文字层级（Text hierarchy）

| 层级 | Token | 色值 | 用途 |
|------|-------|------|------|
| Primary | `--ds-text` | `#111827` | 标题、KPI 数值、表格数据、分类名 |
| Secondary | `--ds-text-secondary` | `#4B5563` | 正文、表头、导航项 |
| Muted | `--ds-text-muted` | `#9CA3AF` | 时间戳、脚注、副标题 **仅此** |

**禁止** 将 Muted 用于主标签或数据本身。

## 3. 字体与排版（Typography）

**字体族**：DM Sans（Display + Body 统一）  
**图标**：Phosphor Icons · Regular · 导航 15–17px · 卡片 16–18px

| 层级 | 规格 | 用法 |
|------|------|------|
| Display | 30–32px / 700 | 极少用 |
| Heading | 20px / 600 · uppercase · letter-spacing | 页面问候、卡片标题 |
| Title | 16px / 600 | 卡片内小节 |
| Body | 14px / 400–500 | 正文、表格、导航 |
| Caption | 12–13px | 元数据、chip 标签、表头 |

**规则**

- 卡片标题、问候语：**大写 + 字间距**（uppercase）
- KPI 数值、表格数据：**sentence case + tabular-nums**
- 表头：12px · uppercase · muted 色

## 5. 布局（Layout）

### 5.1 桌面结构

```
┌──────────┬────────────────────────────────────┐
│ Sidebar  │  Toolbar + Welcome                   │
│ 252px    │  KPI × 4                           │
│          │  ┌──────────────┬─────────┐         │
│          │  │ Wide 1.4     │ Narrow 1│ Row 1   │
│          │  ├──────────────┼─────────┤         │
│          │  │ Wide 1.4     │ Narrow 1│ Row 2   │
│          │  └──────────────┴─────────┘         │
└──────────┴────────────────────────────────────┘
```

### 5.2 Bento 容器高度

| Token | 值 | 适用 |
|-------|-----|------|
| `--ds-panel-row-1-h` | 360px | Source Performance + Source Share |
| `--ds-panel-row-2-h` | 328px | Cost Summary + Cost Breakdown |
| `--ds-panel-content-min` | 248px | 表壳、图表区最小内容高 |
| `--ds-card-head-h` | 44px | 卡片标题行 |
| `--ds-table-row-h` | 40px | 表头 + 表体行 |

**计算**：卡片 ≈ head(44) + body(min 248) + padding(36) → Row1 ≈ 360px

### 5.3 信息密度模式

| 模式 | 场景 | 规则 |
|------|------|------|
| Comfortable | Dashboard 总览 | 卡片 pad 18–20px、行高 40px、控件 36px |
| Compact | 列表/模块（待做） | pad 12–14px、更紧行距 |
| Scrollable | 主列溢出 | 主列滚动；Bento 行高固定 |

## 6. 组件规范（Components）

### 6.1 侧栏品牌 Sidebar Brand

- Mark：36×36 蓝渐变 squircle + `users-three` 图标
- 标题：RecruitOS（uppercase）+ 副标 Vendor & Analytics
- Hover：mark 缩放 1.08 + 微旋转；标题变 brand dark；区块淡 tint
- **无** idle 光晕阴影

### 6.2 侧栏导航 Sidebar Nav

- 结构：`<button>` + Phosphor 图标 + 文案；Badge 右对齐
- 图标容器：28×28 圆角底块（icon well）
- **Active**：品牌 tint 底 + icon well 高亮，**无左侧竖线、无投影**
- **Inactive hover**：淡品牌底 + icon well tint + icon 微放大
- 分组标签：Main / Insights / Admin（12px uppercase muted）

### 6.3 Welcome 卡片

- 全宽渐变条 + 左侧太阳图标（44×44 squircle）
- 问候语 uppercase；摘要中关键数字加粗强调色
- Hover：仅边框加深

### 6.4 KPI 卡片

结构：**图标+标签行 → 大数值 → 趋势 Tag + 对比脚注**

| 字段 | 样式 |
|------|------|
| 图标 | 36×36 圆角底 |
| 数值 | 大号、primary 色 |
| Tag | Teal / Coral / Amber / Blue 语义色 |
| Meta | muted · 如 "vs. 6 last period" |

Hover：边框 tint + 上浮 1px

### 6.5 数据表格 Source Performance

**两层容器**：Card → `ds-table-shell`（12px 圆角 + 1px 边框）

| 列 | 对齐 | 样式 |
|----|------|------|
| Source | 左 | 加粗；hover 变品牌蓝 |
| Resumes / Offers / Hired / Conv. | 右 | `tabular-nums` |
| Hired | 右 | **加粗** |
| Conv. | 右 | 最大值自动高亮品牌蓝 |
| Status | 右 | Tag |

- 表头：tint 底、40px 行高、uppercase muted
- 表体：仅水平分割线，无行级外框
- Row hover：6% 品牌 tint 底
- 手机：横向滚动 + **Source 列 sticky**

### 6.6 环形图 Donut + 图例 Legend

**Donut**

- SVG stroke 弧，非 conic-gradient
- 默认 stroke 5.5px → hover 11.5px（0.34s）
- 非激活扇区 opacity 32%
- 中心 `aria-live` 同步 % 与标签
- **无** hover drop-shadow

**Legend 行**

- 布局：dot(8px) + name + % + bar
- Bar track：浅灰 `color-mix(border 72%, surface)`
- Bar fill：对应 chart token 渐变
- Share 卡 4 项 gap 4px；Breakdown 3 项 gap 10px
- 可键盘聚焦，`role="button"`

### 6.7 费用汇总 Cost Summary

**顶部三 Chip**

| Chip | 内容 |
|------|------|
| Paid / Pending | 数字 + uppercase 标签 |
| Total spend | **虚线边框** + 锁图标（金额受限唯一锁位） |

**列表行**

- 8px 色点（与图表色系一致，**无**左侧粗色条）
- Meta：`8 paid · 6 pending` 格式
- 进度条：实色段 = paid，斜纹段 = pending（**非金额**）

**底部**：`Amounts restricted · Request access` 单行提示

### 6.8 Tag 与 Button

**Tag**：Teal / Blue / Amber / Coral 四色胶囊，11–12px

**Button**

- Primary：品牌蓝底白字
- Secondary：白底 + 边框
- Icon btn：36×36 方钮

原型中多数按钮为展示态，不可点击。

### 6.9 语言切换

- 位置：Tab 栏右侧（桌面）/ 手机顶栏
- 默认 **EN**；切换后更新 New Design + 侧栏文案
- Specs / Original **不翻译**

## 7. 响应式（Responsive）

| 断点 | 行为 |
|------|------|
| **> 900px** | 标准桌面：侧栏固定 + Bento 双列 |
| **≤ 900px** | Bento 单列；KPI 2 列；图表区上下堆叠 |
| **≤ 768px** | 手机顶栏 + 抽屉侧栏；表格横滚 + Source sticky；Tab 可横滑 |
| **≤ 480px** | KPI 单列；Welcome 纵向排列 |

手机抽屉：左侧滑出 288px、遮罩、Escape/点击遮罩关闭、打开时锁定 body 滚动。

---

## 8. 无障碍要点（UI 相关）

- 图表图例可键盘操作；donut 中心屏幕阅读器可读
- 受限金额：锁 chip 带 `aria-label`
- 导航：`aria-current="page"` on active
- 装饰性图标：`aria-hidden="true"`
- 减弱动效：见 [动画.md](./动画.md)

## 9. Original vs New Design 对照

| 维度 | Original（原版） | New Design（新设计） |
|------|------------------|----------------------|
| 布局 | 简单四卡网格 | Welcome + KPI + Bento 1.4:1 |
| 侧栏 | 纯文字列表 | 图标 + 分组 + 用户区 |
| 字体 | 系统默认 | DM Sans |
| 配色 | 通用蓝 `#2563eb` | Vivid Sky 体系 |
| 图表 | CSS conic 饼图 | SVG 交互环形图 + 图例 |
| 表格 | 基础表格 | Shell + tint 表头 + 峰值 Conv. |
| 费用 | 授权遮罩整块 | Chip + 分类进度 + 受限提示 |
| 表面 | 轻投影 | 扁平边框 |
| 动效 | 无 | 入场错峰 + 图表/行 hover |
| 语言 | 仅英文 | 中英文切换 |
| 手机 | 侧栏折行 | 抽屉 + sticky 列 |
