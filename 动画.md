# RecruitOS Dashboard — 动效规范（Motion）

> **版本** v1.0 · **适用范围** New Design Tab  
> 面向视觉 / 动效设计师。中文为主，参数保留英文便于与开发对齐。

---

## 1. 动效原则

### 1.1 克制、服务阅读

- 动效 **不抢数据本身**；总览页以静态可读为第一优先
- 反馈以 **底色变化、边框 tint** 为主，少用位移与大投影
- 图表交互强调 **当前扇区**，其余淡化——不引入额外发光

### 1.2 统一时长档位

| 档位 | 时长 | 用途 |
|------|------|------|
| Fast | 0.15s | 表格行、图例行、chip、icon btn |
| Medium | 0.18–0.2s | 导航、KPI 边框、卡片边框 |
| Slow | 0.28–0.34s | 抽屉滑入、环形图 stroke、品牌 mark |
| Entrance | 0.35–0.5s | 页面入场错峰 |

### 1.3 首选缓动

| 名称 | 值 | 用于 |
|------|-----|------|
| ease | 默认 | 大部分 hover / border |
| ease-out 类 | `cubic-bezier(.22, 1, .36, 1)` | 条形入场、抽屉 |

### 1.4 已刻意弱化的效果

以下效果在 v1.0 中 **已移除或不再使用**：

- 卡片 / KPI hover 大投影 elevation
- Logo mark idle 呼吸光晕（`ds-brand-mark-glow`）
- Welcome / 图标阴影脉冲
- Donut slice `drop-shadow`
- 导航 active 左侧竖线 + 投影
- Cost chip hover 上浮 + 阴影

## 2. 页面入场（Entrance）

**触发**：进入 New Design Tab 且用户未开启「减弱动态效果」

**动画**：`ds-fade-up` — `opacity 0→1` + `translateY(10px→0)`

### 2.1 错峰时序（Stagger）

| 元素 | 延迟 | 时长 |
|------|------|------|
| Page toolbar | 0s | 0.35s |
| Welcome 卡片 | 0.06s | 0.40s |
| KPI 1 → 4 | 0.05s → 0.20s（步进 0.05s） | 0.45s |
| Bento 卡 1 → 4 | 0.18s → 0.36s（步进 0.06s） | 0.50s |

### 2.2 条形入场 `ds-bar-in`

- 对象：图例 progress bar fill、费用 paid 进度段
- 效果：`scaleX(0→1)`，origin left
- 时长：0.65s，`cubic-bezier(.22, 1, .36, 1)`，延迟 0.45s

### 2.3 减弱动效（Reduced motion）

```css
@media (prefers-reduced-motion: no-preference) { ... }
```

当系统开启「减弱动态效果」时：

- **全部入场动画关闭**（fade-up、bar-in）
- Welcome 太阳图标 **不脉冲**
- Hover transition 仍保留（瞬时反馈）

## 3. 持续循环动画（Ambient）

| 元素 | 动画 | 周期 | 效果 |
|------|------|------|------|
| Welcome 太阳图标 | `ds-sun-pulse` | 3.5s | 仅 `scale(1→1.04)`，**无阴影变化** |

遵循 `prefers-reduced-motion`：减弱动效时关闭。

## 4. Hover 与 Focus 交互

### 4.1 侧栏导航 Nav item

| 状态 | 效果 |
|------|------|
| Inactive hover | 淡品牌底 + icon well tint + icon `scale(1.03)` |
| Active | 品牌 tint pill + icon well 14% 品牌底 |
| Active hover | 底色略加深，icon `scale(1.04)` |
| Focus | 2px 品牌色 outline |

过渡：0.18s ease

### 4.2 侧栏品牌 Sidebar brand

| 状态 | 效果 |
|------|------|
| Hover | mark `scale(1.08) rotate(-3deg)`；标题 → brand dark；区块 6% tint |

过渡：0.28s `cubic-bezier(.22, 1, .36, 1)`

### 4.3 KPI 卡片

| 状态 | 效果 |
|------|------|
| Hover | 边框 28% 品牌 mix；`translateY(-1px)` |

过渡：0.15s · **无 shadow**

### 4.4 Bento 卡片

| 状态 | 效果 |
|------|------|
| Hover | 边框 22% 品牌 mix |
| 联动 | 内部 chart-well 边框 18% 品牌 mix |

过渡：0.2s · **无 translate、无 shadow**

### 4.5 表格行

| 状态 | 效果 |
|------|------|
| Row hover | 6% 品牌 tint 底；Source 列文字 → 品牌蓝 |

过渡：0.15s

### 4.6 图例行 Legend row

| 状态 | 效果 |
|------|------|
| Hover / slice-hovered | 浅灰底 `--ds-bg` |
| Bar fill | opacity → 0.92 |

### 4.7 费用 Cost

| 元素 | Hover |
|------|-------|
| Cost item 行 | 6% 品牌 tint 底 |
| Cost chip | 边框 tint only |

### 4.8 按钮

| 元素 | Hover |
|------|-------|
| Primary btn | `brightness(1.06)` only |
| Icon btn | 边框 + tint 底 + `scale(1.05)` |

### 4.9 Welcome

| 状态 | 效果 |
|------|------|
| Hover | 边框 → `--ds-welcome-border-hover` |

## 5. 图表动效（Chart）

### 5.1 环形图 Donut

| 属性 | 默认 | Hover（当前扇区） |
|------|------|-------------------|
| stroke-width | 5.5px | 11.5px |
| 其他扇区 opacity | 100% | 32% |
| 过渡 | — | 0.34s `cubic-bezier(.22, 1, .36, 1)` |

**同步行为**

- Hover 扇区或对应图例行 → 中心标签更新 % + 名称
- 图例行获得 `.is-slice-hovered` 背景
- 鼠标离开图表区 → 恢复默认高亮行（`.ds-legend-row--key`）

**无** drop-shadow、无内圆缩放。

### 5.2 图例条形入场

见 §2.2 `ds-bar-in`

## 6. 手机抽屉（Mobile drawer）

| 属性 | 值 |
|------|-----|
| 动画 | `transform translateX(-105% → 0)` |
| 时长 | 0.28s |
| 缓动 | `cubic-bezier(.22, 1, .36, 1)` |
| 遮罩 | opacity 0→1，0.25s |

关闭：Escape、点击遮罩、选导航项后自动关闭。

## 9. 设计验收建议

演示动效时建议：

1. 正常系统设置下打开 New Design，观察入场错峰是否流畅、不拖沓
2. 开启 macOS「减弱动态效果」或 Windows「显示动画」，确认入场消失
3. 环形图逐扇区 hover，检查加粗与淡化是否同步图例
4. 手机宽度打开抽屉，检查滑入是否跟手、无弹跳
