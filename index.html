RecruitOS Design System

v1.0 · Vivid Sky · Clear, professional, subtly energetic

Principles

  • Structured clarity — card boundaries define information zones; avoid flat full-page tables.
  • Vivid Sky palette — production theme: blue ramp for brand & chart segments; one coral-orange accent on the key data point. This doc reflects Vivid Sky only; alternate palettes preview in Themes, not here.
  • Chart interaction — donut rings start thin (5.5px stroke); hovered segment thickens to 11.5px; inactive segments dim to 32% opacity. Center label updates with the active slice; legend stays in sync.
  • Unified hover — brand-tinted background lift on rows, cards, and list items; no lateral slide or border-width jumps.
  • Semantic accents — Teal (positive), Amber (attention), Coral (risk) reserved for status tags — never chart segment fills.
  • Moderate density — comfortable whitespace on overview screens; compact mode for data-heavy modules.
  • Text hierarchy — primary (#111827) for titles, values & data labels; secondary (#4B5563) for body & column headers; muted (#9CA3AF) for timestamps, footnotes & metadata only.

Component · Legend & Progress

Row: dot + name + % + bar. Track = color-mix(border 72%, surface) full-width pill; fill = chart token gradient. Share list gap 4px (4 items); Breakdown gap 10px (3 items). Legend rows: role="button", keyboard focus ring.

Component · Sidebar Brand

RecruitOSVendor & Analytics

Mark: blue gradient squircle 36×36. Hover on brand block: mark scale 1.08 + slight rotate, title → brand dark blue, area tint — no glow shadow. Respects prefers-reduced-motion.

Component · Sidebar Navigation

Items are <button> elements with icon + label; badges right-aligned via margin-left: auto. Icons sit in 28×28 rounded wells. Active: brand tint pill + icon well highlight (no shadow). Inactive hover: light brand wash + icon well tint + slight icon scale. Focus: brand outline ring.

Mobile (≤768px): drawer off-canvas from left; top bar shows hamburger, compact brand, and active module pill synced via data-nav-label. Backdrop tap or Escape closes drawer; body scroll locked while open.

Interaction & Motion

  • Nav items — icon well tint + scale on hover; drawer slide 0.28s
  • Cards / KPI — border tint on hover; KPI translateY −1px
  • Table rows — brand 6–7% background tint
  • Donut — stroke thicken + peer dim; 0.34s ease
  • Cost items — background tint only (no translateX)
  • Entrance — fade-up stagger on New Design view; disabled when reduced-motion

Accessibility

  • Legend rows: focusable, aria-label per slice, Enter/Space activates
  • Donut center: aria-live="polite" for % + label changes
  • Restricted spend: aria-label on top lock chip; progress bars use role="img" + paid/pending label
  • Decorative icons: aria-hidden="true"; meaning carried by visible text or label
  • Sidebar nav: native buttons, aria-current="page" on active item; drawer toggle aria-expanded
  • Display-only prototype: dashboard chrome buttons use tabindex="-1" except nav + doc demos

Responsive · Breakpoints

Below 900px: bento panels stack single column; KPI grid 2 columns (1 column under 480px); chart blocks stack vertically. Below 768px: fixed top bar + off-canvas drawer sidebar (hamburger, backdrop, Escape to close); performance table scrolls horizontally with Source column sticky. Nav items are focusable buttons with icon-well active/hover states; mobile pill shows current module label.

Icons

Phosphor Icons · Regular weight · 15–17px in nav, 16–18px in cards. Single stroke style throughout; no mixed icon sets.
